# Generative AI Playbook

A complete open-source journey from **beginner -> expert** in Generative AI.

This repository is designed for people who want to go from **zero knowledge** to building production-grade AI systems.

The goal is not only learning APIs, but understanding:

- Foundations
- Deep Learning
- Transformers
- LLMs
- RAG
- Agents
- Fine Tuning
- Multimodal AI
- LLMOps
- Research

Each topic contains:

- Theory
- Visual explanations
- Mathematics
- Code implementations
- Production concepts
- Exercises
- Projects

---

# Learning Path

Beginner

Expert
